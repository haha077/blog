Typora 常用快捷键

  1. Ctrl + / 切换编辑语法模式/预览模式

  2. Ctrl + 1 6 标题 16 等同于

    一级标题：#+Space+标题内容

    二级标题：##+Space+标题内容

    三级标题：###+Space+标题内容

    ……

    最多六级标题

  3. Ctrl + B 粗体

    这是粗体

  4. Ctrl + I 斜体

    这是斜体

  5. Ctrl + T 表格

    1 2 3
    4 5 6

  6. Ctrl + U 下划线

    下划线

  7. Alt+Shift+5 或 *内容* 删除线

    删除线

  8. — 或*** 分割线

    分割线

  9. ```+Enter Ctrl + Shift + K代码块

    #include <bits/stdc++.h>
using namespace std;

    CopyC++

  10. $$

    • Enter 或 Ctrl + Shift + M公式块

    $$

    \varphi = 1+\frac{1} {1+\frac{1} {1+\frac{1} {1+\cdots} } } $$

  11. 引用：>+Space 或 Ctrl+Shift+Q

  12. 转义字符： \ + 字符

  13. 无序列表：-+ Space 或 * + Space 或 Ctrl + Shift + ]

  14. 有序列表：0（数字） + . + Space 或 Ctrl + Shift + [

  15. 生成目录：[toc] + Enter

  16. 插入链接：Ctrl+K

  17. 表情：:+单词+:

    常用表情地址
